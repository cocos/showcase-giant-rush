## 游戏玩法
**奔跑吧巨人**是一款冒险闯关游戏，通过收集途中的同颜色小人壮大自己，冲过重重机关，
击败守关 BOSS。过关同时收集钻石，可购买解锁更多皮肤效果。

### 操作说明
* 游戏中巨人会自动前进，玩家需要滑动屏幕控制巨人左右移动。
* 当进入战斗时根据提示滑动可躲避敌人攻击，点击则可攻击敌人。
